1.Запуск без аргументов: node index.js
  Вызывается справка и передается код 1.

2. Запуск с недостаточным количеством аргументов: 
   node index.js 1 2
   Выход с ошибкой: недостаточное количество аргументов и кодом ошибки 2.

3. Запуск с аргументами node index.js 1 6 у
   Вывод в консоль: Ошибка: все аргументы должны быть числами.
   Передается код 2.

4. Запуск с аргументами node index.js 0 3 5
   Вывод в консоль: Ошибка: коэффициент a не может быть равен нулю (уравнение не квадратное).
   Передается код 2.
   
5. Запуск с аргументами node index.js 2 3 5
   Вывод в консоль: Действительных корней нет (дискриминант отрицательный).
   Передается код 2.

6. Запуск с аргументами node index.js 1 6 9
   Вывод в консоль: Уравнение имеет один корень: x = -3
   Передается код 0.

7. Запуск с аргументами node index.js 22 33 5.
   Вывод в консоль: Уравнение имеет два корня: x₁ = -0.17101185441559108, x₂ = -1.328988145584409
   Передается код 0.
